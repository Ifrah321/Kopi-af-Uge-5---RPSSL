Brugeren vælger en shape via knapper: Rock, Paper, Scissors, Spock eller Lizard.

Agenten vælger en tilfældig shape ved hjælp af Random.

Programmet viser:

spillerens valg

agentens valg

resultatet af runden

den løbende score

Når en af parterne når 3 point, vises vinderen, og scoren nulstilles.

Der bruges enum til shapes og resultat, og en lille funktion afgør vinderen ud fra RPSSL-reglerne.

UI opdateres efter hver runde via simple TextBlock-felter.

Koden følger undervisningens stil: code-behind, ingen MVVM og helt basale Avalonia-kontroller.

Der skal afleveres en kort screencast (<10 sek.) der viser, at spillet virker og UI opdateres.
